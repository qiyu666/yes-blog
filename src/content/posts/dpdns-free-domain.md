## 注册前准备

开始注册前，你需要准备好以下材料，以便顺利完成整个流程：

- **GitHub 账号**：没有的话请先访问 GitHub 官网注册一个。
- **一个常用邮箱**：用于接收验证邮件，QQ 邮箱或 Gmail 等均可。
- **个人信息生成工具（可选）**：注册 dpdns 平台时可能需要填写格式正确的英文地址和电话号码，若不清楚格式，可借助"身份大全"等工具网站生成。

## 注册步骤

### 第一步：注册 dpdns 平台账户

访问 dpdns 平台的注册页面：https://dash.domain./auth/register

填写注册信息：

- **用户名**：自选一个用户名。
- **英文姓名**：按格式要求填写即可。
- **邮箱**：填写你准备的可用邮箱。
- **电话号码**：需符合格式（例如 +1-1234567890）。
- **英文地址**：确保地址中包含两个英文逗号，这是通过格式验证的关键。
- **密码**：设置你的账户密码。
- **人机验证**：完成验证操作。

点击 Register 提交注册。

### 第二步：验证邮箱

注册完成后，到你填写的邮箱中查收验证邮件（可能有一到两封）。点击邮件中的验证链接完成邮箱验证。

### 第三步：登录并完成 KYC（身份验证）

使用刚才注册的账户登录 dpdns 平台：https://dash.domain./auth/login

首次登录后，系统通常会提示你进行身份验证（KYC）。请选择 GitHub OAuth 方式。

页面将跳转至 GitHub 授权，点击 Authorize EdwardLab 完成授权关联。

### 第四步：注册域名

KYC 验证通过后，你就可以开始注册域名了：

访问域名注册页面：https://dash.domain./panel/main?page=%2Fpanel%2Fregister

在输入框中输入你心仪的域名前缀（例如 mytestdomain），后缀选择 .dpdns.org。

勾选同意相关协议，点击 Check Availability 检查域名是否可用。

如果域名可用，页面会跳转。按提示填写两个 NS 服务器地址（可先随意填写，例如 ns1.example.com 和 ns2.example.com，后续在域名管理中能修改）。

点击 Register!，恭喜你，域名就注册成功了！

### 第五步：获取更多免费域名（可选）

默认情况下，一个账户可能只能注册一个免费域名。但你可以通过以下方法增加名额：

1. 访问 GitHub 上的指定项目：https://github.com/DigitalPlatDev/FreeDomain
2. 点击右上角的 Star 按钮收藏该项目。
3. 然后访问 https://dash.domain./auth/login/github 进行授权。

成功后，你的账户通常会获得额外的注册名额（例如+1）。之后便可重复第四步注册更多域名。

## 域名的配置与管理

### 修改 DNS 服务器（以 Cloudflare 为例）

注册域名后，建议将其托管到 Cloudflare 等专业 DNS 服务商，以便使用更强大的解析、CDN 或邮件路由等功能。

#### 获取 Cloudflare Nameserver 地址

1. 注册并登录 Cloudflare。
2. 添加你的站点（输入你的 xxx.dpdns.org 域名），选择免费计划。
3. 按照指引，Cloudflare 会提供两个名称服务器地址（如 xxx.ns.cloudflare.com 和 yyy.ns.cloudflare.com），记下它们。

#### 在 dpdns 平台修改 NS 记录

1. 登录 dpdns 平台，进入域名管理页面：https://dash.domain./panel/main?page=%2Fpanel%2Fdomains
2. 找到你的域名并点击管理。
3. 将 NS 记录修改为从 Cloudflare 获取到的两个名称服务器地址。
4. 保存更改。DNS 解析完全生效可能需要几十分钟到几小时，请耐心等待。

### 配置邮件路由（可选）

将域名托管到 Cloudflare 后，你还可以免费开启 Catch-all 邮件路由功能，实现无限别名邮箱。

1. 在 Cloudflare 控制面板中选择你的域名。
2. 找到 Email 或 Email Routing 选项并进入。
3. 点击 Enable Email Routing 启用邮件路由。

#### 配置 Catch-all 地址

1. 启用 "Catch-all Address" 功能。
2. 设置一个常用的邮箱（如你的 Gmail 地址）作为转发目标。
3. 到目标邮箱查收 Cloudflare 发送的验证邮件并点击验证。

验证成功后，发往 任意前缀@你的域名.dpdns.org 的邮件都会被自动转发到你指定的邮箱。

## 注意事项

- **域名可用性**：心仪的域名前缀可能已被注册，需多准备几个备选。
- **信息填写**：注册 dpdns 平台时，英文地址的格式（包含两个英文逗号）和电话号码的格式（如 +1-1234567890）要符合要求，否则可能无法通过验证。
- **域名续期**：dpdns.org 免费域名需要每180天续期一次（有来源提到有效期可能为一年，但强调需在过期前180天内续期）。请密切关注邮箱通知，或在账户后台查看到期时间并及时续期，以免域名失效。
- **服务稳定性**：免费域名服务可能存在不确定性（例如之前的 us.kg 域名就因滥用等问题停止了免费注册）。它非常适合用于学习、测试和非关键项目，但不建议用于非常重要的生产环境或商业项目。
- **邮件管理**：开启 Catch-all 功能后，虽然方便，但也可能收到发往你域名的所有垃圾邮件，请确保目标邮箱的垃圾邮件过滤功能正常。
